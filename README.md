# RetailConnect

RetailConnect is a web-based platform designed to connect retailers and wholesalers through a digital marketplace. The platform integrates an AI-powered chatbot using IBM Watson Assistant to provide instant support, business information, and supplier discovery.

## Technologies Used
- HTML
- CSS
- JavaScript
- IBM Watson Assistant
- IBM Cloud Platform

## Features
- Supplier discovery system
- Retailer–wholesaler collaboration
- AI-powered chatbot assistance
- Business information support
- Simple and responsive web interface
